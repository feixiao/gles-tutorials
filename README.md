# OpenGL ES 2.0 Tutorials
> version 0.1.0

## <a name="description"></a>Description
本教程基于OpenGL ES（以下简称gles）2.0，基本内容以示例程序的形式编写。因为它是基于在QNX上操作的，所以需要了解初始化信息，但由于其他所有内容都是基于gles API编写的，因此可以从任何平台借用。
